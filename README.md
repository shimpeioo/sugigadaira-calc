# 杉ヶ平キャンプ場 料金計算機

21世紀の森 杉ヶ平キャンプ場（NPO 大長谷村づくり協議会）の現場スタッフ用 料金計算ツール。

## 概要

- スタッフが現場で利用料金をミスなく計算するためのモバイル対応ツール
- 仕様書 v0.2 §5.5.2 / §5.7 に基づく
- ホスティング：GitHub Pages（無料）
- PWA対応：iPadのホーム画面に追加してアプリのように使える、オフラインでも動作

## 使い方（iPad mini 2 など）

1. Safari で公開URLを開く
2. 共有ボタン →「ホーム画面に追加」
3. 以降はアイコンタップで起動（オフラインOK）

## ローカル動作確認

```bash
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000/` を開く。

## 料金マスタ

仕様書 v0.2 §5.5.2 を参照：

- コテージ A棟（6人用）：13,200円／1泊（基本6人、追加+1,100円/人で最大8人）
- コテージ B/C/D/E棟（4人用）：8,800円／1泊（基本4人、追加+1,100円/人で最大6人。E棟のみ追加不可）
- バンガロー：3,150円／1泊（人数制限なし）
- テント貸出：880円／張、テント持込：660円／張
- キャンプ場利用人数（テント時）：220円／人（幼児無料）
- 連泊：2泊目以降すべて半額
- 日帰り：宿泊料金の半額

## ファイル構成

```
index.html       UIと計算ロジック
manifest.json    PWA設定
sw.js            Service Worker（オフラインキャッシュ）
icon-192.png     アイコン（192×192）
icon-512.png     アイコン（512×512）
```

## 更新方法

料金変更などがあったときは、コードを修正して `git push` するだけ。
次回起動時にService Workerが自動的に新バージョンを取得。
